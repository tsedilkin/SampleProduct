# Инструкция по запуску проекта

## Шаг 1: Установка Node.js

Если у вас не установлен Node.js:
1. Перейдите на https://nodejs.org/
2. Скачайте и установите LTS версию
3. Перезапустите терминал

## Шаг 2: Установка зависимостей

Откройте терминал в папке проекта и выполните:
```bash
npm install
```

## Шаг 3: Запуск проекта

После установки зависимостей запустите dev-сервер:
```bash
npm run dev
```

## Шаг 4: Открытие в браузере

После запуска команды `npm run dev` вы увидите что-то вроде:
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

Откройте в браузере адрес: **http://localhost:5173/**

## Альтернативный способ (если не хотите устанавливать Node.js)

Можно открыть файл `index.html` напрямую в браузере, но для работы React нужен сервер разработки.
